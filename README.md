# Sum Speed

Sum Speed is a Python script for comparing the time used in completing different types of sums. 

| Method                    | Time [s]  |
| ------------------------- | -----     |
| while loop                | 4.992     |
| for pure                  | 3.169     |
| sum range                 | 2.384     |
| sum generator             | 5.881     |
| sum list comp             | 5.171     |
| numpy sum                 | 0.180     |
| numpy sum python range    | 7.344     |
| math sum                  | 1.600e-6  |
